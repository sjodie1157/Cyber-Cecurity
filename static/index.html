These are the paths for manipulating the database

Static Page
HTTP method Path Function of the path
NONE / Displays the static page

Users

HTTP method Path Function of the path
get /Users Returns all users in an array of objects
post /users Adds a new user
get /users/:id Returns a single user by id
delete /users/:id Deletes a single user by id
patch /users/:id Edits a single user by id

Items

HTTP method Path Function of the path
get /Items Returns all Items in an array of objects
post /Items Adds a new product
get /Items/:id Returns a single product by id
delete /Items/:id Deletes a single product by id
patch /Items/:id Edits a single product by id
